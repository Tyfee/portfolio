My Expertise

Hello there! I'm a web developer with over five years of freelancing experience, building a variety of web applications using the latest technologies. In addition to my work in web development, I've been involved in software and game development/design, contributing to projects at Spacelog Games as a programmer, 3D modeler, pixel art artist, and soundtrack producer.

Frontend

In frontend web development i mainly work with the React and Vue frameworks, though i have participated in projects using Svelte as well.

Backend

In backend web development i mainly work with Ruby, Python and Jvascript environments, though Rust and Go are also languagues that i am accostumed to use for that purpose. My go to frameworks are FastAPI, Express and Sinatra.
